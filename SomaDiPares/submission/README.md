# Submissão: Soma de Pares consecutivos

Arquivo pronto para submissão ao beecrowd (URI).

Como compilar e executar:

```bash
# compilar
javac Main.java

# executar (leia entradas até 0)
java Main

# exemplo de uso (stdin):
printf "4\n11\n0\n" | java Main
```

Descrição rápida: o programa lê inteiros até encontrar `0`. Para cada inteiro lido imprime a soma dos 5 pares consecutivos a partir do valor (inclui o valor se ele for par).
